电阻电路的一般分析详解

3.1 电路的图

核心概念

  1. 图（Graph）

    • 定义：由支路（边）和节点（顶点）组成的拓扑结构，用于描述电路的连接关系。
    • 特性：
      • 支路和节点一一对应电路中的元件和连接点。
      • 孤立节点允许存在（移除支路不改变节点存在性）。

  2. 路径与连通图

    • 路径：支路的连续序列，连接两个节点。
    • 连通图：任意两节点间至少存在一条路径；否则为非连通图。

  3. 子图与树（Tree）

    • 子图：原图的部分支路和节点组成的图。
    • 树：满足以下条件的子图：
      • 连通；
      • 包含所有节点；
      • 无闭合路径（即无回路）。
    • 树支：构成树的支路；连支：非树支的支路。
    • 树支数：n-1（n为节点数）；连支数：b - (n-1)（b为支路数）。

  4. 基本回路（单连支回路）

    • 每个连支与树支构成的唯一回路，独立回路数为连支数。
    • 平面电路中，网孔数等于独立回路数。

3.2 KCL和KVL的独立方程数

KCL的独立方程数

  • 规则：对于n个节点的电路，独立KCL方程数为n-1。
  • 原因：最后一个节点的KCL方程可由其他方程推导得出。

KVL的独立方程数

  • 规则：独立KVL方程数等于基本回路数，即：
    b - (n-1)（b为支路数）。
  • 平面电路：网孔数即独立KVL方程数。

3.3 支路电流法

方法概述

  • 变量：各支路电流（共b个变量）。
  • 方程数：n-1个KCL方程 + b - (n-1)个KVL方程 = b个独立方程。

步骤详解

  1. 标定参考方向：为所有支路电流和电压设定参考方向。
  2. 列写KCL方程：选择n-1个节点，列写其电流守恒方程。
  3. 列写KVL方程：选择独立回路（通常为网孔），结合欧姆定律将电压转换为电流表达式。
  4. 求解方程：解线性方程组得到支路电流。
  5. 后续分析：计算支路电压、功率等。

3.4 网孔电流法

方法概述

  • 适用条件：平面电路。
  • 变量：网孔电流（假想的沿网孔闭合流动的电流）。
  • 方程数：网孔数（等于独立回路数）。

方程列写规则

  1. 自电阻（R_ii）：网孔内所有电阻之和，总为正。
  2. 互电阻（R_ij）：
    • 若两网孔电流方向相同，取正；否则取负。
  3. 电压源处理：
    • 方向与网孔电流一致时，电压源取负；否则取正。

3.5 回路电流法

方法概述

  • 适用条件：平面和非平面电路。
  • 变量：独立回路电流（需选择独立回路）。
  • 方程数：b - (n-1)个KVL方程。

核心技巧

  • 灵活选择独立回路：使电流源支路仅属于一个回路，减少方程数。
  • 处理受控源：先视为独立源，后用回路电流表示控制量。

与网孔电流法的对比

对比项 网孔电流法 回路电流法
适用范围 平面电路 所有电路
回路选择 必须为网孔 任意独立回路
方程复杂度 互电阻符号规则简单 需灵活处理互电阻

3.6 节点电压法

方法概述

  • 变量：节点电压（参考节点外，共n-1个变量）。
  • 方程数：n-1个KCL方程。

方程列写规则

  1. 自电导（G_ii）：节点连接的所有支路电导之和，总为正。
  2. 互电导（G_ij）：节点间所有公共支路电导之和，总为负。
  3. 电流源处理：流入节点取正，流出取负。

特殊处理

  • 无伴电压源：
    • 方法1：引入电流变量，增补电压约束方程。
    • 方法2：选择电压源一端为参考节点，简化方程。
  • 受控源：先用节点电压表示控制量，再代入方程。

方法对比与选择

方法 适用场景 优点 缺点
支路电流法 支路数少、需直观分析 直接、易于理解 方程数多（b个）
网孔电流法 平面电路、网孔数少 方程数少（网孔数） 仅限平面电路
回路电流法 非平面电路、含多电流源 灵活选择回路 需处理复杂互电阻
节点电压法 节点数少、含多电压源 方程数少（n-1个） 需处理无伴电压源

总结

  • 核心思想：通过拓扑分析（图论）和基尔霍夫定律，系统化减少方程数量。
  • 关键技巧：
    • 选择合适的变量（电流或电压）以降低复杂度。
    • 灵活处理特殊元件（电流源、受控源）的约束条件。
  • 实际应用：根据电路特点（平面性、节点/支路数量）选择最优方法。
