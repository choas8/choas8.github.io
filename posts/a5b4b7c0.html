电路基础

第一章 电路模型与基本定律

1.1 实际电路 vs 电路模型

  • 实际电路：为完成某种预期目的而设计、制作、运行，由电路部件和电路器件相互连接而成的电流通路装置
    • 功能分类：
      • 强电电路：能量的传输、转换、分配、使用。
      • 弱电电路：信息的传递、控制、处理。
  • 电路模型：用理想元件（电阻、电源）抽象表示实际电路，便于分析。
    • 核心概念：
      • 电源：提供电能（如电池）。
      • 负载：消耗电能的设备（如灯泡）。
      • 激励：电源产生的电压/电流。
      • 响应：电路中由激励产生的电压/电流。
      • 输入：根据激励与响应的关系，把激励称为输入。
      • 输出：根据激励与响应的关系，把响应称为输出。

1.2 电流与电压的参考方向

  • 电流方向：假设一个参考方向，计算结果的正负决定实际方向。
    • 示例：若电流参考方向为从左到右，计算得 I = -2A，则实际方向为从右到左。
  • 电压方向：假设从“+”极指向“-”极的参考方向。
  • 关联方向：电流与电压参考方向一致时为关联方向，否则为非关联方向。

提示

  • 电路中电位参考点可任意选择；
  • 参考点一经选定，电路中各点的电位值就唯一确定；
  • 当选择不同的电位参考点时，电路中各点电位值将改变，但任意两点间电压保持不变。


1.3 电功率与能量

  • 功率公式：P = 电压 × 电流
    • 关联方向下：P > 0 表示元件吸收功率，P < 0 表示发出功率。
  • 能量计算：能量 = 功率 × 时间（如灯泡工作1小时消耗 W = P × 3600秒）。

1.4 电路元件详解


1.4.1 电阻元件

  • 欧姆定律：电压 = 电阻 × 电流（关联方向下成立）。
  • 特殊情况：
    • 开路：电阻无穷大，电流为0。
    • 短路：电阻为0，电压为0。
  • 功率计算：P = I²R = U²/R（电阻总是吸收功率）。

1.4.2 理想电源

  • 电压源：输出电压恒定，电流由外电路决定（如干电池）。
    • 特性：不可短路（否则电流无限大）。
  • 电流源：输出电流恒定，电压由外电路决定（如恒流芯片）。


1.4.3 受控电源

  • 定义：输出量受电路中另一处电压/电流控制。
  • 四种类型：
    类型 控制关系 示例
    VCVS 输出电压 ∝ 控制电压 晶体管放大器
    CCVS 输出电压 ∝ 控制电流 传感器信号转换
    VCCS 输出电流 ∝ 控制电压 场效应管
    CCCS 输出电流 ∝ 控制电流 双极型晶体管

1.5 基尔霍夫定律

1.5.1 基尔霍夫电流定律（KCL）

  • 内容：流入任一节点的电流总和等于流出电流总和。
    • 示例：若 I₁=2A，I₂=3A，则流出电流 I₃ = I₁ + I₂ = 5A。


1.5.2 基尔霍夫电压定律（KVL）

  • 内容：闭合回路中所有电压的代数和为零。
    • 应用步骤：
      1. 设定回路绕行方向（顺时针或逆时针）。
      2. 沿方向累加电压：电源电压与绕行方向一致取负，相反取正。

第二章 电阻电路的等效变换

2.1 等效电路概念

  • 等效条件：两个电路的端口电压-电流关系（VCR）相同。
  • 目的：简化复杂电路，方便计算。

2.2 电阻的串联与并联

2.2.1 串联电阻

  • 总电阻：R_eq = R₁ + R₂ + ... + Rₙ
  • 分压公式：Uₖ = (Rₖ / R_eq) × 总电压

2.2.2 并联电阻

  • 总电导：G_eq = G₁ + G₂ + ... + Gₙ
  • 分流公式：Iₖ = (Gₖ / G_eq) × 总电流







2.3 Y-Δ（星型-三角型）变换

  • 适用场景：复杂桥接电路无法直接串并联时使用。
  • 变换公式：
    • Y→Δ：R_Δ = (R₁R₂ + R₂R₃ + R₃R₁) / Rₖ
    • Δ→Y：R_Y = R_Δ相邻电阻乘积 / ΣR_Δ



2.4 电源等效变换

2.4.1 电压源与电流源的互换

  • 电压源→电流源：I_S = U_S / R，并联电阻R。
  • 电流源→电压源：U_S = I_S × R，串联电阻R。

2.4.2 实际电源模型

  • 实际电压源：理想电压源 + 串联内阻。
  • 实际电流源：理想电流源 + 并联内阻。







2.5 输入电阻计算

  • 定义：从端口看进去的等效电阻。
  • 方法：
    1. 串并联法：直接化简电阻网络。
    2. 加压求流法：端口加电压U，测电流I，R_in = U / I。
